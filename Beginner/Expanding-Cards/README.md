## Descrição do Projeto

Este projeto consiste em uma página HTML que demonstra o funcionamento de cartões expansíveis. Cada cartão contém uma imagem e um título relacionado a um local. Ao clicar em um cartão, ele se expande para ocupar mais espaço na tela. Além disso, há um botão que permite alternar entre os layouts de coluna e linha, proporcionando uma experiência de usuário diferenciada. O código inclui tanto HTML quanto CSS para a estilização, e também JavaScript para a interatividade dos cartões e o controle do layout.
