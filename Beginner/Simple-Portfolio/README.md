## Descrição do Projeto

Este é um projeto de um site de portfólio pessoal desenvolvido em HTML. Ele apresenta uma estrutura simples, contendo um cabeçalho com o nome do desenvolvedor e links para suas redes sociais, uma seção principal com informações sobre o desenvolvedor e suas habilidades em tecnologias como HTML, CSS, JavaScript e Java. Além disso, o site inclui uma seção de contato com um formulário para os visitantes entrarem em contato. O projeto também possui um rodapé com informações de direitos autorais.

## Tecnologias Utilizadas

   ```

2. Clone o repositório:
   S

   cd Simple-Portfolio
   ```
