## Descrição do Projeto

Este projeto consiste em uma página HTML para uma "Agenda Telefônica". Ele permite que os usuários cadastrem contatos com seus nomes e números de telefone. A página também exibe uma tabela onde os contatos são listados. Cada linha da tabela contém o nome e o número do contato, junto com um botão para exclusão.

O formulário de cadastro valida se ambos os campos (nome e telefone) foram preenchidos corretamente antes de adicionar um novo contato. O número de telefone deve ser inserido no formato "(##) #####-####". Caso o formato não seja seguido, uma mensagem de alerta será exibida.
